==================
dolmen.forms.table
==================

``dolmen.forms.table`` is a extension to `dolmen.forms.base`_ to create CRUD
forms: it let define define forms that can work on a collection of
content sharing the same form schema.

As well ``dolmen.forms.table`` can be embedded as a sub form in a
composed form built with `dolmen.forms.composed`_.

.. _dolmen.forms.base: http://pypi.python.org/pypi/dolmen.forms.base
.. _dolmen.forms.composed: http://pypi.python.org/pypi/dolmen.forms.composed
